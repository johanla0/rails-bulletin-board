### Instructions
#### Install locally (no docker containers currently)

`make install`

#### Start a new branch

`make branch feature_name`

#### Bundle js and watch the stimulus controllers directory

`make esbuild`

#### Check styles (slim, js, scss) and run tests (models, controllers) before pushing

`make check`

#### Push

`make push`
